Project Description

This project aims to develop a robust machine learning model for forecasting PM2.5 air pollution in Tashkent. The model is built upon a comprehensive dataset created by fusing multiple disparate sources, including ground-level sensor readings, ERA5 atmospheric reanalysis data, satellite observations of aerosols (MODIS) and trace gases (Sentinel-5P), active fire data from NASA FIRMS, and static geographical information. The methodology involved a rigorous data cleaning pipeline, featuring advanced imputation techniques such as K-Nearest Neighbors (KNN) to handle missing sensor data, ensuring a high-quality, model-ready dataset for accurate air quality prediction.

Dataset Overview

  • Data Type: CSV (Comma-Separated Values)
