Project Description

In this project, we will analyze and build predictive models using the Automobile Dataset. This dataset, publicly available online, provides comprehensive information about various car attributes, such as price, engine performance, and fuel efficiency. The goal is to explore the data, preprocess it, and develop regression models to predict car prices based on key features.

Dataset Overview

  • Data Source: Automobile Dataset
  • Data Type: CSV (Comma-Separated Values)

Key Objectives

  1. Part-1: Clean and preprocess the dataset, handling missing values and converting categorical variables into quantitative forms.
  2. Part-2: Investigate relationships between features and the target variable (price) using Exploratory Data Analysis (EDA).
  3. Part-3: Develop regression models, including Linear Regression, Multiple Linear Regression, and Polynomial Regression.
  4. Part-4: Compare models and refine them using techniques like Cross-Validation and Ridge Regression.

Tools and Libraries

  • Python Libraries: pandas, numpy, matplotlib, seaborn, scikit-learn
  • Visualization Tools: Matplotlib and Seaborn for data and model visualization
